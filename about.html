Cloud Development Experience

Most of my career as a software dev has been devoted to cloud migration. For a developer at a large company, this trained me to function as a jack-of-all-trades. I know enough about database organization and SQL to implement secure ETL and quering solutions; I have a strong web-dev background that allows me work comfortably in React, Angular, Vue, or Svelte codebases; I have a solid understanding of API layers, business logic, infrastructure-as-code, and AWS cloud services. And since my team's software accesses live data including protected health information, I'm well versed in cloud security and zero-trust authentication and authorization services, including those offered by Hashiorp, Azure, and AWS.
